#    영문으로 작성하는 경우 동사(원형)을 가장 앞에 작성함
#    과거시제는 사용하지 않음

#    <Body> 규칙
#    본문은 72글자 이내로 작성

#    여러줄 작성 시 "-"로 구분

#    <Footer> 규칙
#    Fixes	    : 이슈 수정중 (아직 해결되지 않은 경우)

#    Ref	    : 참고할 이슈가 있을 경우 사용
#    Related to	: 해당 커밋에 관련된 이슈번호 (아직 해결되지 않은 경우)
#    ex) Fixes: #46 Related to: #32, #35
